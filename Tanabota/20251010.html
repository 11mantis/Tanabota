昨日「メソッド(JavaScript)を勉強中なんですね。今お話しされている内容を聞いて太一さんがどんなレベルなのかがわかりました。勉強頑張ってください。」と言われる事がありました。
言ってきた本人には”下のレベルの奴にマウントを取ってやろう”なんて意図は無いのかもしれないけど、言われた人(自分)はそれなりに気にするなと思う事がありました。

まぁ、、、js少ししか理解できていないのは事実だし。
「はい、頑張ります。」としか言いようがありません。

◎関数：「足し算をしてください」という内容の、addという名前の枠。
◎変数：a + bの答え(結果)を受け止めるための受け皿(今回の場合はc)。
◎引数：関数に渡す値(今回だとa, b)。
◎戻り値：returnで返ってきた結果(今回だとc)。
function add (a, b) {
const c = a + b
return c
}
_
const c = add(2, 3)
console.log(c)
_
順番としては、(2, 3)→関数 add()→戻り値(5)→console.log(5)
こんな流れです。
